# 🤝 Contributing

This isn't a code repo, but there are plenty of ways to help:

## ⭐ Star This Repo
Seriously — it helps us show up in GitHub search. One click, big impact.

## 📊 Submit Data
Have a clean dataset of U.S. businesses (CSV/JSON)? Open an issue describing:
- Source and licensing
- Format and fields
- Geographic coverage
- Approximate record count

## 🐛 Report Data Issues
Found a wrong address, closed business, or duplicate? Open an issue or use the Report button on the listing page.

## 📣 Spread the Word
- Share with AI agent builders
- Tell local business owners to claim their listing
- Write about us (we'll link back)
- Add our MCP server to your agent

## 🔧 Build on Our API
Built something cool with our API? Open an issue and we'll feature it in USE-CASES.md.

## 💡 Suggest Features
Open an issue tagged `feature-request`. We read everything.
